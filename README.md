# Brain-games

Mini-games, that ask you the simple questions

# Setup

   sudo make install -g brain-games_mygame

# Run

The game to determine even and odd numbers

    brain-even

Example: https://asciinema.org/a/Fndlz5tAWfALh5Uq3mUUH4H1s
